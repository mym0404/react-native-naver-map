React Native Naver Map의 전체 API 문서입니다.

# 주요 컴포넌트



## NaverMapView



지도를 표시하는 메인 컴포넌트입니다.

```tsx
interface NaverMapViewProps {
  // 기본 설정
  style?: ViewStyle;
  initialRegion?: Region;
  mapType?: MapType;
  layerGroups?: LayerGroups;
  
  // UI 컨트롤
  isShowCompass?: boolean;
  isShowScaleBar?: boolean;
  isShowZoomControls?: boolean;
  isShowLocationButton?: boolean;
  isShowIndoorLevelPicker?: boolean;
  
  // 지도 설정
  isIndoorEnabled?: boolean;
  isNightModeEnabled?: boolean;
  isLiteModeEnabled?: boolean;
  isExtentBoundedInKorea?: boolean;
  
  // 스타일
  symbolScale?: number;
  lightness?: number;
  logoAlign?: LogoAlign;
  logoMargin?: Point;
  
  // 제약
  minZoom?: number;
  maxZoom?: number;
  
  // 이벤트
  onInitialized?: () => void;
  onCameraChanged?: (event: CameraChangedEvent) => void;
  onOptionChanged?: (event: OptionChangedEvent) => void;
  onTapMap?: (event: MapTapEvent) => void;
}
```



## NaverMapMarkerOverlay

```tsx
interface NaverMapMarkerOverlayProps {

  subCaption?: MarkerCaption;

  isHideCollidedSymbols?: boolean;
  isHideCollidedMarkers?: boolean;
  isHideCollidedCaptions?: boolean;
  isForceShowIcon?: boolean;
  isForceShowCaption?: boolean;

  minZoom?: number;
  maxZoom?: number;

  children?: React.ReactNode;

```

# 타입 정의



## Region



지도 영역을 정의합니다.

```tsx
interface Region {
  latitude: number;      // 중심 위도
  longitude: number;     // 중심 경도
  latitudeDelta: number; // 위도 범위
  longitudeDelta: number; // 경도 범위
}
```



## Coord



좌표를 나타냅니다.

```tsx
interface Coord {
  latitude: number;  // 위도
  longitude: number; // 경도
}
```



## Point



화면상의 점을 나타냅니다.

```tsx
interface Point {
  x: number; // X 좌표 (0.0 ~ 1.0)
  y: number; // Y 좌표 (0.0 ~ 1.0)
}
```



## MapType



지도 유형을 정의합니다.

```tsx
type MapType = 
  | 'Basic'     // 기본 지도
  | 'Navi'      // 내비게이션 지도
  | 'Satellite' // 위성 지도
  | 'Hybrid'    // 하이브리드 지도
  | 'Terrain';  // 지형도
```



## MarkerImage



마커 이미지 타입을 정의합니다.

```tsx
type MarkerImage = 
  | { symbol: MarkerSymbol }           // 기본 심볼
  | ImageSourcePropType                // 로컬 이미지
  | { assetName: string }              // 네이티브 리소스
  | { httpUri: string };               // 네트워크 이미지

type MarkerSymbol = 
  | 'red' | 'green' | 'blue' | 'yellow' 
  | 'gray' | 'pink' | 'lightblue';
```



## MarkerCaption



마커 캡션을 정의합니다.

```tsx
interface MarkerCaption {
  text: string;           // 텍스트
  textSize?: number;      // 텍스트 크기
  color?: string;         // 텍스트 색상
  haloColor?: string;     // 테두리 색상
  requestedWidth?: number; // 요청 너비
  minZoom?: number;       // 최소 줌
  maxZoom?: number;       // 최대 줌
}
```



## LayerGroups



레이어 그룹 설정을 정의합니다.

```tsx
interface LayerGroups {
  BUILDING?: boolean;   // 건물
  TRAFFIC?: boolean;    // 교통 정보
  TRANSIT?: boolean;    // 대중교통
  BICYCLE?: boolean;    // 자전거 도로
  MOUNTAIN?: boolean;   // 등산로  
  CADASTRAL?: boolean;  // 지적도
}
```

# 이벤트 타입



## CameraChangedEvent



카메라 변경 이벤트 데이터입니다.

```tsx
interface CameraChangedEvent {
  latitude: number;   // 위도
  longitude: number;  // 경도
  zoom: number;       // 줌 레벨
  tilt: number;       // 기울기
  bearing: number;    // 회전각
  reason: CameraChangeReason; // 변경 이유
}

type CameraChangeReason = 
  | 'gesture'   // 사용자 제스처
  | 'control'   // UI 컨트롤
  | 'location'  // 위치 추적
  | 'program';  // 프로그래밍
```



## MapTapEvent



지도 탭 이벤트 데이터입니다.

```tsx
interface MapTapEvent {
  latitude: number;  // 탭한 위치의 위도
  longitude: number; // 탭한 위치의 경도
}
```

# Ref 메서드



## NaverMapViewRef



NaverMapView 컴포넌트의 참조를 통해 호출할 수 있는 메서드들입니다.

```tsx
interface NaverMapViewRef {
  // 카메라 제어
  animateCameraTo(params: CameraAnimationParams): void;
  animateRegionTo(region: Region, duration?: number): void;
  setCameraTo(params: CameraParams): void;
  setRegionTo(region: Region): void;
  
  // 위치 추적
  setLocationTrackingMode(mode: LocationTrackingMode): void;
  
  // 유틸리티
  takeSnapshot(options?: SnapshotOptions): Promise<string>;
  screenToCoordinate(point: { x: number; y: number }): Promise<Coord>;
  coordinateToScreen(coord: Coord): Promise<{ x: number; y: number }>;
}
```



## CameraAnimationParams



카메라 애니메이션 매개변수입니다.

```tsx
interface CameraAnimationParams {
  latitude?: number;   // 목표 위도
  longitude?: number;  // 목표 경도
  zoom?: number;       // 목표 줌 레벨
  tilt?: number;       // 목표 기울기
  bearing?: number;    // 목표 회전각
  duration?: number;   // 애니메이션 시간 (ms)
}
```



## LocationTrackingMode



위치 추적 모드입니다.

```tsx
enum LocationTrackingMode {
  None = 0,      // 추적 안함
  NoFollow = 1,  // 위치만 표시
  Follow = 2,    // 위치 추적
  Face = 3,      // 위치 + 방향 추적
}
```

# 유틸리티 함수



## 거리 계산



두 좌표 간의 거리를 계산합니다.

```tsx
import { getDistance } from '@mj-studio/react-native-naver-map';

const distance = getDistance(
  { latitude: 37.5665, longitude: 126.9780 },
  { latitude: 37.5670, longitude: 126.9785 }
);
console.log(`거리: ${distance}m`);
```



## 좌표 변환



화면 좌표와 지도 좌표 간 변환입니다.

```tsx
// 지도 좌표를 화면 좌표로 변환
const screenPoint = await mapRef.current?.coordinateToScreen({
  latitude: 37.5665,
  longitude: 126.9780
});

// 화면 좌표를 지도 좌표로 변환
const coord = await mapRef.current?.screenToCoordinate({
  x: 100,
  y: 200
});
```
