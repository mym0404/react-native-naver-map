## Overview

NaverMapCircleOverlay는 지도 위에 원형 오버레이를 표시하는 컴포넌트입니다.
중심점 좌표, 반지름, 색상 등을 설정하여 원형 영역을 표시할 수 있습니다.

## Basic Usage

```tsx
import { NaverMapView, NaverMapCircleOverlay } from '@mj-studio/react-native-naver-map';

function MapWithCircle() {
  return (
    <NaverMapView style={{ flex: 1 }}>
      <NaverMapCircleOverlay // [!code highlight:7]
        latitude={37.5665}
        longitude={126.9780}
        radius={1000}
        color={'#ff00ff33'}
        outlineColor={'#ff00ff'}
        outlineWidth={2}
      />
    </NaverMapView>
  );
}
```
