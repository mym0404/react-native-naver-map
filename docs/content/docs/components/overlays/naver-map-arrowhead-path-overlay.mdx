## Overview

NaverMapArrowheadPathOverlay는 지도 위에 화살표가 포함된 경로 오버레이를 표시하는 컴포넌트입니다.
경로의 방향을 나타내는 화살표를 포함하여 시각적으로 방향성을 표현할 수 있습니다.

## Basic Usage

```tsx
import { NaverMapView, NaverMapArrowheadPathOverlay } from '@mj-studio/react-native-naver-map';

function MapWithArrowPath() {
  return (
    <NaverMapView style={{ flex: 1 }}>
      <NaverMapArrowheadPathOverlay // [!code highlight:12]
        coords={[
          { latitude: 37.5, longitude: 126.9 },
          { latitude: 37.51, longitude: 126.91 },
          { latitude: 37.52, longitude: 126.92 },
        ]}
        headSizeRatio={2.5}
        width={8}
        color={'red'}
        outlineColor={'blue'}
        outlineWidth={2}
        onTap={() => console.log('Arrow path tapped')}
      />
    </NaverMapView>
  );
}
```
