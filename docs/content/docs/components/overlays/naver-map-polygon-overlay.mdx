## 개요

NaverMapPolygonOverlay는 지도 위에 폴리곤(다각형) 오버레이를 표시하는 컴포넌트입니다.
좌표 배열을 통해 다각형 영역을 정의하고, 채우기 색상과 외곽선 스타일을 설정할 수 있습니다.
