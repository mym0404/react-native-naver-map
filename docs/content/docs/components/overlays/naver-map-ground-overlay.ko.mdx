## Overview

NaverMapGroundOverlay는 지도 위에 그라운드 오버레이를 표시하는 컴포넌트입니다.
특정 지리적 영역에 이미지를 오버레이하여 표시할 수 있습니다.

## Basic Usage

```tsx
import { NaverMapView, NaverMapGroundOverlay } from '@mj-studio/react-native-naver-map';

function MapWithGroundOverlay() {
  return (
    <NaverMapView style={{ flex: 1 }}>
      <NaverMapGroundOverlay // [!code highlight:9]
        image={{ assetName: 'thumbnail' }}
        region={{
          latitude: 37.5,
          longitude: 126.9,
          latitudeDelta: 0.1,
          longitudeDelta: 0.1,
        }}
        onTap={() => console.log('Ground overlay tapped')}
      />
    </NaverMapView>
  );
}
```
