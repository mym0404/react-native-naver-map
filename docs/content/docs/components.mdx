## 기본 사용법

```tsx
import {
  NaverMapView,
  NaverMapMarkerOverlay,
  NaverMapCircleOverlay
} from '@mj-studio/react-native-naver-map';

function MapScreen() {
  return (
    <NaverMapView style={{ flex: 1 }}>
      {/* 마커 추가 */}
      <NaverMapMarkerOverlay
        latitude={37.5665}
        longitude={126.9780}
        anchor={{ x: 0.5, y: 1 }}
      />

      {/* 원형 오버레이 추가 */}
      <NaverMapCircleOverlay
        latitude={37.5665}
        longitude={126.9780}
        radius={100}
        color="#ff000033"
      />
    </NaverMapView>
  );
}
```

## 컴포넌트 계층 구조

모든 오버레이 컴포넌트는 `NaverMapView`의 자식으로 사용되어야 합니다:

```tsx
<NaverMapView>
  <NaverMapMarkerOverlay />
  <NaverMapCircleOverlay />
  <NaverMapPolygonOverlay />
  {/* 기타 오버레이들... */}
</NaverMapView>
```

## 마커 최적화



대량의 마커를 표시할 때는 다음 사항을 고려하세요:

1. **이미지 캐싱**: `reuseIdentifier` 속성 활용
2. **뷰포트 기반 렌더링**: 보이는 영역의 마커만 표시
3. **클러스터링**: 근접한 마커들을 그룹화

## TypeScript 지원

모든 컴포넌트는 완벽한 TypeScript 타입을 제공합니다:

```tsx
import type {
  NaverMapViewProps,
  Region,
  Coord,
  MapType
} from '@mj-studio/react-native-naver-map';

const initialRegion: Region = {
  latitude: 37.5665,
  longitude: 126.9780,
  latitudeDelta: 0.01,
  longitudeDelta: 0.01,
};

const mapProps: NaverMapViewProps = {
  initialRegion,
  mapType: 'Basic',
  isShowLocationButton: true,
};
```
