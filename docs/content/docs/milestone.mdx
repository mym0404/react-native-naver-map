### Location Overlay Command Implementation

Implement imperative API control for Location Overlay manipulation.

<Tabs items={['Feature Overview', 'Implementation Details']}>
<Tab value="Feature Overview">
Location Overlay is a special overlay that displays the user's current location on the map. Only one exists per map and can be controlled through imperative commands.

```typescript
// Basic location and visibility control
naverMapRef.current?.locationOverlay.setVisible(true);
naverMapRef.current?.locationOverlay.setPosition({
  latitude: 37.5665,
  longitude: 126.9780
});
naverMapRef.current?.locationOverlay.setBearing(45); // Direction angle

// Icon customization
naverMapRef.current?.locationOverlay.setIcon({
  source: require('./user-location.png'),
  width: 24,
  height: 24,
  anchor: { x: 0.5, y: 0.5 }
});

// Accuracy circle display
naverMapRef.current?.locationOverlay.setCircleRadius(100);
naverMapRef.current?.locationOverlay.setCircleColor('#4285F4');
naverMapRef.current?.locationOverlay.setCircleOutlineWidth(2);
naverMapRef.current?.locationOverlay.setCircleOutlineColor('#FFFFFF');
```
</Tab>

- Integrate with React Native Fabric architecture
- Create test cases in example app
</Tab>

<Tabs items={['Package Separation', 'API Design', 'Distribution Strategy']}>

<Tab value="API Design">
Wrap REST API-based services with React Native-friendly Promise/async-await patterns:

```typescript
import { NaverCloudServices } from '@react-native-naver-map/cloud-services';

// Directions API
const route = await NaverCloudServices.directions({
  start: { lat: 37.5665, lng: 126.9780 },
  goal: { lat: 37.5651, lng: 126.9895 },
  option: 'trafast'
});

// Geocoding API
const coordinates = await NaverCloudServices.geocoding({
  query: 'Seoul Jung-gu Sejong-daero 110'
});
```

## Detailed API Plans

### Directions API

Integrate route search and navigation functionality into React Native apps.

```typescript
interface DirectionsOptions {
  start: Coordinate;
  goal: Coordinate;
  waypoints?: Coordinate[];
  option?: 'trafast' | 'tracomfort' | 'traoptimal';
  avoid?: ('toll' | 'ferry')[];
}

interface DirectionsResult {
  route: {
    trafast: RouteInfo[];
    tracomfort: RouteInfo[];
    traoptimal: RouteInfo[];
  };
}
```

### Geocoding API

Service for converting addresses to coordinates.

```typescript
interface GeocodingOptions {
  query: string;
  coordinate?: Coordinate; // Search center point
  filter?: string; // Search filter
}

interface GeocodingResult {
  addresses: Address[];
  meta: {
    totalCount: number;
    page: number;
    count: number;
  };
}
```

### Reverse Geocoding

Service for converting coordinates to addresses.

```typescript
interface ReverseGeocodingOptions {
  coords: Coordinate;
  output?: 'json' | 'xml';
  orders?: string;
}

interface ReverseGeocodingResult {
  results: AddressComponent[];
}
```

### Additional Services

Other Naver Cloud Platform APIs planned for future addition:

- **Static Map API**: Generate static map images
- **Places API**: Place search and information lookup
- **Traffic API**: Real-time traffic information
