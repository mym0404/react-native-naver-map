## Expo 설정 (Config Plugin)

Expo 프로젝트에서 React Native Naver Map을 사용하기 위한 설정 방법을 안내합니다.

<Callout type="warn">
Expo Go는 네이티브 모듈을 지원하지 않으므로 Development Build가 필요합니다.
</Callout>

## 1. expo-build-properties 패키지 추가

이는 네이버 Maven 저장소를 주입하기 위한 것입니다.

```bash
npx expo install expo-build-properties
```

## 2. app.json에 Config Plugin 추가

발급받은 Client ID와 함께 Config Plugin을 설정합니다:

```json title="app.json"
{
  "expo": {
    "name": "your-app-name",
    "plugins": [
      [
        "@mj-studio/react-native-naver-map",
        {
          "client_id": "YOUR_CLIENT_ID_HERE",
          "android": {
            "ACCESS_FINE_LOCATION": true,
            "ACCESS_COARSE_LOCATION": true,
            "ACCESS_BACKGROUND_LOCATION": true
          },
          "ios": {
            "NSLocationAlwaysAndWhenInUseUsageDescription": "이 앱은 지도에서 현재 위치를 표시하기 위해 위치 정보를 사용합니다.",
            "NSLocationWhenInUseUsageDescription": "이 앱은 지도에서 현재 위치를 표시하기 위해 위치 정보를 사용합니다.",
            "NSLocationTemporaryUsageDescriptionDictionary": {
              "MapDisplay": "지도에서 현재 위치를 표시하기 위해 위치 정보를 사용합니다."
            }
          }
        }
      ],
      [
        "expo-build-properties",
        {
          "android": {
            "extraMavenRepos": ["https://repository.map.naver.com/archive/maven"]
          }
        }
      ]
    ]
  }
}
```

<Callout type="warn">
`YOUR_CLIENT_ID_HERE` 부분을 네이버 클라우드 플랫폼에서 발급받은 실제 Client ID로 교체해주세요.
</Callout>

<Callout type="info">
위치 권한 설정은 선택사항입니다. expo-location 패키지를 사용하여 별도로 설정할 수도 있습니다.
</Callout>

## 3. Prebuild 실행

네이티브 프로젝트 파일을 생성합니다:

```bash
npx expo prebuild -p android
npx expo prebuild -p ios
```

## 4. Development Build 생성

Expo Go는 네이티브 모듈을 지원하지 않으므로 Development Build가 필요합니다:

```bash
## iOS
npx expo run:ios

## Android
npx expo run:android
```

또는 EAS Build를 사용하여 클라우드에서 빌드할 수 있습니다:

```bash
## EAS Build 설정
npx eas build:configure

## Development Build 생성
npx eas build --profile development --platform ios
npx eas build --profile development --platform android
```

## 5. 개발 서버 실행

Development Build가 완료되면 개발 서버를 시작합니다:

```bash
npx expo start --dev-client
```

<Callout type="success">
Expo 설정이 완료되었습니다! Development Build에서 네이버 지도를 사용할 수 있습니다.
</Callout>

## 문제 해결

## 빌드 오류가 발생하는 경우

1. **Android 빌드 오류**: `android/build.gradle`에 Maven 저장소가 올바르게 추가되었는지 확인
2. **iOS 빌드 오류**: `ios/Podfile`을 삭제하고 다시 prebuild 실행
3. **Config Plugin 오류**: app.json 문법이 올바른지 확인

## Metro 설정 오류

```javascript title="metro.config.js"
const { getDefaultConfig } = require('expo/metro-config');

const config = getDefaultConfig(__dirname);

module.exports = config;
```
