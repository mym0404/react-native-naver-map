# Configuration

There are three ways to configurate `expo-release-it`

## 1. via cli options

```shell
npx expo-release-it build --platform android --androidOutput aab
```

The `--platform` and `--androidOutput` options are used for skipping prompt during execution.

Every commands are powered by [Commander.js](https://github.com/tj/commander.js#readme) which means you can `-h` or `--help` to view all of available options.

```shell
npx expo-release-it upload --help
```

## 2. via prompt during execution

The **required options not passed from cli options** nor config file are prompted during execution using [`Inquirer.js`](https://github.com/SBoudrias/Inquirer.js)

Therefore, you don't need any extra configuration or to consider cli options before running commands.

## 3. via `expo-release-it.config.json`

put `expo-release-it.config.json` into your project root or generated `expo-release-it` directory.

Or you can provide config file path with `--config` cli option in any command.
