# React Native Naver Map

React Native 앱에 네이버 지도를 통합할 수 있는 강력하고 유연한 라이브러리입니다.

<Callout type="info" emoji="🚀">
  **v2.x 버전**부터는 React Native의 New Architecture (Fabric)만 지원합니다.
  Bridge 아키텍처를 사용 중이라면 v1.x 버전을 사용해주세요.
</Callout>

## 주요 특징

### 🏗️ New Architecture 지원
React Native 0.74+ 버전의 Fabric 렌더러를 완벽하게 지원합니다. 네이티브 성능과 동기식 레이아웃으로 더욱 부드러운 지도 경험을 제공합니다.

### 📚 상세한 문서화
모든 컴포넌트와 API에 대한 완벽한 TypeScript 타입 정의와 JSDoc 주석을 제공합니다. IDE에서 자동 완성과 인라인 문서를 통해 개발 생산성을 높일 수 있습니다.

### 📱 Expo 지원
Expo Config Plugin을 통해 Expo 프로젝트에서도 손쉽게 사용할 수 있습니다. Development Build와 Production 환경 모두 지원합니다.

### ⚡ 최적화된 마커 성능
다양한 마커 타입을 지원하며, 이미지 캐싱과 최적화를 통해 수천 개의 마커도 부드럽게 렌더링합니다.

### 🔄 네이티브 SDK 완벽 호환
최신 네이버 지도 SDK (iOS/Android 3.22.1)의 모든 기능을 React Native에서 사용할 수 있도록 포팅했습니다.

## 빠른 시작

```tsx
import { NaverMapView, NaverMapMarkerOverlay } from '@mj-studio/react-native-naver-map';

export default function MapScreen() {
  return (
    <NaverMapView
      style={{ flex: 1 }}
      initialRegion={{
        latitude: 37.5665,
        longitude: 126.9780,
        latitudeDelta: 0.01,
        longitudeDelta: 0.01,
      }}
    >
      <NaverMapMarkerOverlay
        latitude={37.5665}
        longitude={126.9780}
        anchor={{ x: 0.5, y: 1 }}
        caption={{ text: "서울시청" }}
      />
    </NaverMapView>
  );
}
```

<Callout type="warning">
  네이버 지도를 사용하려면 [네이버 클라우드 플랫폼](https://www.ncloud.com)에서 Maps API 키를 발급받아야 합니다.
</Callout>

## 버전 호환성

| React Native Naver Map | React Native | 비고 |
|---|---|---|
| ≥ 2.1.0 | ≥ 0.74 | Bridge 지원 중단, New Architecture 필수 |
| ≥ 2.0.0 | ≥ 0.73 | New Architecture 전용 |
| < 2.0.0 | ≥ 0.64 | Old/New Architecture 모두 지원 |

<Callout>
  2025년 4월 17일부터 [새로운 Maps 단독 상품](https://www.ncloud.com/support/notice/all/1930)이 출시되었습니다.
  기존 AI NAVER API 고객은 레거시 키 설정을 참고해주세요.
</Callout>
